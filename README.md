# Sistema Academico

## Capas

- `src/presentacion`: controladores de entrada.
- `src/aplicacion`: servicios de aplicacion y casos de uso.
- `src/dominio`: entidades, value objects, factories, servicios de dominio e interfaces de repositorio.
- `src/infraestructura`: implementaciones concretas de repositorios.
- `docs/uml`: archivo UML original.
- `tests`: pruebas del proyecto.
